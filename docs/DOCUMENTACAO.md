# Sistema de Agendamento

## Problema
Pequenos prestadores de serviço realizam agendamentos manualmente,
o que gera conflitos de horário.

## Objetivo
Desenvolver um sistema web para controle de agendamentos.

## Requisitos Funcionais
- Cadastro de usuários
- Login
- Agendamento de horários

## Regras de Negócio
- Não permitir horários duplicados
